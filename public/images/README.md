# Image Placeholders

This app uses placeholder divs for images. To add real images:

1. Place your images in this `/public/images/` folder
2. Update the image references in the code
3. For menu item photos, upload to Supabase Storage and store the URL in the `photo` column

Recommended: Use Supabase Storage for menu item images.

Background image for login/signup: `bg-placeholder.jpg`
Logo: Add your logo as `logo.jpg` or update the component.
